Teting task from Alawar company.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

В электронной фильмотеке кинотеатра на каждый день есть N фильмов различных жанров. 
Каждый фильм может принадлежать нескольким жанрам. 
В каждом зале кинотеатра могут транслироваться только фильмы определенных жанров. 
Необходимо создать каталог фильмов и составить расписание на день для каждого из залов кинотеатра на основе каталога. 
Каждый фильм должен быть показан максимальное кол-во раз, но в каждом зале показан не больше 1 раза за день.

Для определенности возьмем 10 фильмов (movie1 ... movie10), 7 жанров (janre1 ... janre7) и 5 кинозалов (hall1 ... hall5).

Movie1: janre1, janre3
Movie2: janre2, janre4
Movie3: janre1, janre3, janre5
Movie4: janre4, janre5, janre7
Movie5: janre3
Movie6: janre1, janre5
Movie7: janre6, janre3, janre2
Movie8: janre3, janre2, janre4
Movie9: janre6, janre2
Movie10: janre2, janre4

Hall1: janre1
Hall2: janre2, janre3
Hall3: janre3, janre5
Hall4: janre4, janre2, janre6
Hall5: janre1, janre5, janre6, janre7

Задание должно быть выполнено на языке С++ в ООП стиле.
